唯一合法合规使用ChatGPT的方法 ​

引言

总所周知，OpenAI目前已经禁止国内用户访问chagpt，而目前能使用的都是通过使用VPN来实现访问的；使用VPN不仅不合法合规，而且也很不稳定，经常触发人机校验，而且最新的gpt-4o也有次数限制。下面介绍唯一一种可以合法合规使用chatgpt的方式。

前提准备 ​

  1. 一张国内银行的国际信用卡，比如 VISA 这种，没有的可以办理；
  2. 一个企业邮箱（随便到网易企业邮箱注册一个就行）

1.Azure 注册教程 ​

第一步：访问 Azure 官网 ​

  1. 打开浏览器，访问 Azure 官网。
  2. 点击页面右上角的 “开始免费” 或 “注册”。

第二步：创建 Microsoft 账户 ​

  1. 如果你没有 Microsoft 账户，点击 “创建一个！”。
  2. 填写所需信息，如电子邮件地址、密码等。
  3. 按照屏幕提示完成账户创建。

第三步：验证身份 ​

  1. 根据提示输入你的手机号码或备用邮箱。
  2. 收到验证码后，输入验证码进行验证。

第四步：填写个人信息 ​

  1. 填写个人或公司信息，包括名称、国家/地区等，地址最好写香港的，可以使用这个地址生成器 https://www.meiguodizhi.com/hk-address
  2. 确认信息的准确性后，点击 “下一步”。

第五步：选择订阅计划 ​

  1. 你可以选择 “免费试用” 或其他付费计划(付费的话就选择随用随付)。
  2. 如果选择免费试用，确保了解提供的免费额度。

第六步：输入付款信息 ​

  1. 输入信用卡信息以验证身份（不会立即收费）。
  2. 按照提示完成付款信息的填写。

第七步：阅读并接受条款 ​

  1. 阅读服务条款和隐私政策。
  2. 勾选同意框并点击 “注册”。

第八步：完成注册 ​

  1. 注册成功后，你将被重定向到 Azure 管理门户。
  2. 你可以开始创建和管理 Azure 资源。

小贴士 ​

  • 确保使用有效的电子邮件和手机号码，以便接收验证信息。
  • 定期检查 Azure 的更新和新功能，以充分利用平台资源。

参考链接 ​

  • Azure 文档
  • Azure 免费帐户

2.创建订阅 ​

第一步：创建一条订阅（一般进去会有默认的一条，有就不用创建） ​

点击主页的订阅，没有就搜一下

第二步：绑定付款信息 ​

打开你刚刚注册的微软账户，付款选项，绑定付款，可以支付宝

第三步 申请Azure Openai ​

申请要填写的内容比较多，你需要用到你注册的企业邮箱和第一步中的订阅ID

耐心等待结果，我差不多12个小时就申请通过了

第四步 使用Openai ​

申请成功后回到Azure面板 https://portal.azure.com/#home 在Azure服务中找到Azure Openai，开始创建资源，不同区域提供的模型不一样，可以查看Azure Openai服务区域定价
