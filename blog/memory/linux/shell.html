Linux 常用命令 ​

文件和目录操作 ​

1. 列出目录内容 ​

bash
ls
1
  • ls：列出当前目录下的文件和目录。
  • ls -l：以长格式显示详细信息。
  • ls -a：显示所有文件，包括隐藏文件。

2. 切换目录 ​

bash
cd /path/to/directory
1
  • cd：切换到指定目录。
  • cd ..：返回上级目录。
  • cd ~：切换到当前用户的主目录。

3. 创建目录 ​

bash
mkdir directory_name
1
  • mkdir：创建一个新目录。

4. 删除目录 ​

bash
rmdir directory_name
1
  • rmdir：删除空目录。
  • rm -r directory_name：递归删除目录及其内容。

5. 删除文件 ​

bash
rm file_name
1
  • rm：删除指定文件。
  • rm -f file_name：强制删除文件。

6. 复制文件 ​

bash
cp source_file destination_file
1
  • cp：复制文件或目录。
  • cp -r source_directory destination_directory：递归复制目录。

7. 移动文件 ​

bash
mv source_file destination_file
1
  • mv：移动或重命名文件或目录。

8. 查看文件内容 ​

bash
cat file_name
1
  • cat：显示文件内容。
  • less file_name：分页显示文件内容。
  • head file_name：显示文件的前部分内容。
  • tail file_name：显示文件的后部分内容。

系统信息 ​

1. 查看当前目录 ​

bash
pwd
1
  • pwd：显示当前工作目录的路径。

2. 查看系统时间 ​

bash
date
1
  • date：显示当前系统时间和日期。

3. 查看系统信息 ​

bash
uname -a
1
  • uname -a：显示系统的详细信息。

4. 查看内存使用情况 ​

bash
free -h
1
  • free -h：以人类可读的格式显示内存使用情况。

5. 查看磁盘使用情况 ​

bash
df -h
1
  • df -h：以人类可读的格式显示磁盘空间使用情况。

6. 查看磁盘空间 ​

bash
du -sh /path/to/directory
1
  • du -sh：显示目录或文件的磁盘使用情况。

进程管理 ​

1. 查看当前进程 ​

bash
ps aux
1
  • ps aux：显示所有进程的信息。

2. 查看进程树 ​

bash
pstree
1
  • pstree：以树形结构显示进程。

3. 终止进程 ​

bash
kill process_id
1
  • kill：发送信号终止指定进程。
  • kill -9 process_id：强制终止进程。

4. 查看进程使用的资源 ​

bash
top
1
  • top：动态显示进程的实时资源使用情况。

网络管理 ​

1. 查看网络配置 ​

bash
ifconfig
1
  • ifconfig：显示网络接口的配置信息。
  • ip a：现代替代 ifconfig，显示网络接口的配置信息。

2. 查看网络连接 ​

bash
netstat -tuln
1
  • netstat -tuln：显示所有活动的网络连接和监听端口。

3. 测试网络连通性 ​

bash
ping hostname
1
  • ping：测试与主机的网络连通性。

4. 查看端口占用 ​

bash
lsof -i :port_number
1
  • lsof -i :port_number：查看指定端口的占用情况。

文件权限 ​

1. 查看文件权限 ​

bash
ls -l file_name
1
  • ls -l：显示文件的权限、所有者和其他详细信息。

2. 修改文件权限 ​

bash
chmod permissions file_name
1
  • chmod：修改文件或目录的权限。例如，chmod 755 file_name。

3. 修改文件所有者 ​

bash
chown user:group file_name
1
  • chown：修改文件或目录的所有者和组。

包管理 ​

1. 更新包列表 ​

bash
sudo apt update
1
  • sudo apt update：更新 Debian/Ubuntu 系统的包列表。

2. 安装软件包 ​

bash
sudo apt install package_name
1
  • sudo apt install：安装指定的软件包。

3. 卸载软件包 ​

bash
sudo apt remove package_name
1
  • sudo apt remove：卸载指定的软件包。

4. 更新所有包 ​

bash
sudo apt upgrade
1
  • sudo apt upgrade：升级系统中所有已安装的软件包。
